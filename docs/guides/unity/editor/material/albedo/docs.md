# Albedo

- 물체가 가진 고유한 색과 반사율을 정의

## Base Map

- 'Albedo'와 동일한 개념의 다른 라벨
    - 'Unity'의 렌더링 파이프라인 및 에디터 버젼에 따라 라벨이 바뀜

- 표준 세이더(Built-in Render Pipeline):
    - 'Albedo'로 표기
- Universal Render Piepline(URP)
    - 'Base Map'이라고 표기
