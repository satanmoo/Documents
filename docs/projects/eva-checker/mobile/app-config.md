# App config

## 앱 이름 바꾸기

- app.config.js

- name:
    - 앱 이름
    - 디바이스에 표시됨
- slug:
    - Expo 프로젝트에서 가시적인 이름
    - URL 에서 사용됨
        - 소문자, 하이픈만 사용
- ios.bundleIdentifier:
    - iOS 앱의 고유 식별자
    - 일반적으로 com.<company>.<appname> 형식으로 설정
- android.package:
    - Android 앱의 고유 식별자
    - 일반적으로 com.<company>.<appname> 형식으로 설정

- 앱 이름에 맞게 위 필드 수정하면 됨
